# Advent of Code 2025

In [mojo](https://docs.modular.com/mojo/manual/install/).

## Developer Setup

Initial setup:

```shell
uv init --app
uv add mojo --default-index https://dl.modular.com/public/nightly/python/simple/ --prerelease allow
mojo --version
```

Update:

```shell
uv sync --upgrade-package mojo
```
